웹 문서에 새로운 이미지 노드 추가하기 

        속성 값이 있는 노드 추가하기. 
        img 태그 뿐 아니라 태그 안의 속성인 src 속성도 넣어줘야한다. 
        HTML 태그 만들 때는 관련된 속성 노드도 함께 만들어서
        자식 노드로 연결해야한다.
        1. img 태그 요소 노드 생성: createElement() 메서드
        2. 속성 노드 생성: createAttribure() 메서드
        3. 속성 노드 연결: setAttributeNode() 메서드
        4. 자식 노드 연결: appendChild() 메서드
